# npm-loot-generator-system
A handy loot generator system for game design

### step 1

- create npm project
- test npm project
- publish npm project

### step 2

- create cli project
- test cli project with `hello world` output
- import your published npm project into cli project
- connect cli function to npm package function for get loot
- test cli project on cli
